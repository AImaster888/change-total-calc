# 工程契約變更簽辦計算系統

工程契約變更簽辦計算系統，提供公共工程契約變更作業使用，協助辦理：

- 契約追加減金額計算
- 新增項目試算
- 承商利潤與營業稅驗證
- 契約變更金額彙算
- 計算歷程輸出
- 簽辦附件生成
- DOCX/PDF 匯出

本系統以「工程契約變更簽辦自動化」為核心目標，並依工程實務及公務行政流程設計。

---

# 系統特色

## 1. 契約變更金額自動計算

系統可依輸入資料自動計算：

| 項目 | 說明 |
|---|---|
| A1 | 原契約金額 |
| A2 | 原契約工項追加金額 |
| A3 | 原契約工項追減金額 |
| A4 | 新增項目金額 |
| A5 | 承商利潤及保險費 |
| A6 | 職業安全衛生費 |
| A7 | 材料檢試驗費 |
| B1 | 變更項目總和 |
| B2 | 變更總金額 |
| B3 | 營業稅 |
| B4 | 本次變更金額 |
| B5 | 變更後契約總金額 |

# 驗證機制

系統內建工程契約變更常用驗證規則。

## 利潤率檢核

- 追加時：
  - 承商利潤不得超過包工費 10%
- 追減時：
  - 應合理扣減承商利潤

---

## 營業稅檢核

- 營業稅以變更總金額 5% 為原則
- 追加不得超給
- 追減宜多扣

---

## 正負值檢核

追加時：

- 012
- 013
- 014

應為正值。

追減時：

- 012
- 013
- 014

應為負值。

# 新增項目試算

系統提供新增項目含稅試算功能。

包含：

- 新增項目佔比
- 012 承商利潤分攤
- 013 職安衛分攤
- 014 材料檢試驗分攤
- 營業稅計算
- 含稅總價計算

---
